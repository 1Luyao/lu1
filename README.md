# Research data
## meteorological data
The meteorological data used in this study were obtained from the CRU TS 4.09 (Climatic Research Unit gridded Time Series, Version 4.09) dataset, jointly released by the Climatic Research Unit (University of East Anglia) and the Met Office.
The dataset is accessible via the official website: https://crudata.uea.ac.uk/cru/data/hrg/cru_ts_4.09/ (this link is for use only when access through BADC is unavailable). 
Detailed descriptions of the dataset are documented in the literature:
Harris, I., Osborn, T.J., Jones, P. et al. Version 4 of the CRU TS monthly high-resolution gridded multivariate climate dataset. Sci. Data 7, 109 (2020). https://doi.org/10.1038/s41597-020-0453-3.
